Claves de Elección Teoría

No existe "el mejor chatbot": existe el más adecuado para cada tarea. La elección correcta depende de: acceso a Internet en tiempo real, ventana de contexto y coste.

⚠️

Privacidad: Ningún chatbot gratuito garantiza que tus conversaciones no se usen para entrenamiento. No pegues datos personales ni información confidencial.

Práctica Guiada — Benchmark personal Práctica
  1. Selecciona una tarea concreta de tu asignatura (ej: "Resume la Segunda Revolución Industrial en 5 puntos").
  2. Envía el mismo prompt a tres chatbots en tres pestañas simultáneas.
  3. Evalúa las respuestas en base a: precisión factual, extensión, fuentes y legibilidad.
  4. Registra en Google Sheets: Chatbot | Puntuación (1-5) | Observaciones.
  5. Concluye qué chatbot resuelve mejor esa tarea. Cambia de chatbot para tareas distintas.
Prompt Maestro — Benchmark de chatbots Prompt
benchmark-chatbot.txt
# BENCHMARK DE CHATBOTS — PROMPT ESTANDARIZADO Actúa como un experto en [materia o tema específico]. Tu tarea es [describe la tarea de forma clara y acotada]. Requisitos: - Responde en español, lista numerada, máximo 5 puntos. - Cada punto: máximo 2 frases. - Si incluyes datos o fechas, cítalos con precisión. - Al final: nivel de confianza [Alto / Medio / Bajo] y por qué. # Envía este mismo prompt a Gemini, ChatGPT y DeepSeek. # Compara las respuestas y anota diferencias en contenido y estilo.
