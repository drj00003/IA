Explicación Teórica Teoría

Gamma transforma un esquema textual (outline) en una presentación completa con diseño, tipografías e imágenes. El flujo es: Prompt → Outline generado → Deck editado → Exportación.

PowerPoint + Copilot (Microsoft 365) permite generar slides desde un documento Word, refinar contenido con prompts dentro de la app y auto-diseñar layouts con Designer. Requiere licencia M365.

💡

El error más común es pedir "haz una presentación sobre X" sin contexto. Especifica siempre: audiencia, duración, número de slides y tono. Cuanto más específico, más utilizables serán las diapositivas.

Práctica Guiada — Gamma Práctica
  1. Crea cuenta gratuita en gamma.app. El plan free incluye 400 créditos iniciales.
  2. "Generar" → "Presentación" → "Generar desde un prompt".
  3. Introduce el prompt detallado (usa la estructura del Prompt Maestro). Gamma generará primero un outline editable.
  4. Revisa y edita el outline antes de generar el deck. Este paso es crítico: añade, elimina o reordena secciones según necesites.
  5. Selecciona el tema visual (paleta + tipografía) y genera el deck completo.
  6. Edita slide por slide: haz clic en cualquier elemento. Pide a la IA del panel lateral que "reescriba este párrafo en tono más técnico".
  7. Exporta en PDF o PowerPoint (.pptx) para entregar o seguir editando.
Prompt Maestro — Gamma Prompt
gamma-presentation-prompt.txt
# PROMPT MAESTRO PARA GAMMA.APP Crea una presentación profesional sobre [TEMA]. Detalles: - Audiencia: [estudiantes de bachillerato / clientes / directivos] - Duración estimada: [10 minutos] - Número de slides: [8-10] - Tono: [formal y técnico / divulgativo / motivador] - Idioma: español Estructura requerida: 1. Portada con título impactante y subtítulo 2. Índice / Agenda 3. [Sección 1 — concepto clave] 4. [Sección 2 — datos o estadísticas] 5. [Sección 3 — ejemplos o casos de uso] 6. [Sección 4 — ventajas y limitaciones] 7. Conclusiones en 3 puntos clave 8. Slide de cierre con llamada a la acción Estilo: moderno, minimalista, con iconos y datos visualizados. Incluye una imagen relevante por slide siempre que sea posible.
