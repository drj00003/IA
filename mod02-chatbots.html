¿Qué es un GPT? Teoría

Un GPT es un asistente de ChatGPT preconfigurado con un rol, instrucciones permanentes, herramientas y archivos de conocimiento específicos para un dominio concreto. Son gratuitos con cualquier cuenta de ChatGPT.

Técnicamente es una combinación de: system prompt extendido + knowledge files (PDFs propios del creador) + actions (conexiones a APIs externas como Canva o Whimsical), todo preempaquetado y listo para usar.

ℹ️

En el plan gratuito el modelo subyacente es GPT-4o mini, no GPT-4o completo. Los resultados siguen siendo muy útiles para el aula, especialmente en los GPTs con conexión a APIs externas como Canva o Whimsical.

Canva
Infografías, pósters y presentaciones desde lenguaje natural
Gratis▾

Qué hace: Conecta ChatGPT con tu cuenta de Canva. Describes el diseño y el GPT lo genera devolviéndote el enlace para editar. Necesitas vincular tu cuenta Canva (gratuita) la primera vez mediante OAuth.

Caso de uso: Trabajos de investigación con componente visual, proyectos interdisciplinares, entregas que requieran material gráfico profesional sin conocimientos de diseño.

prompt-canva-gpt.txt
# PROMPT PARA CANVA GPT Crea una infografía vertical (A4) sobre [TEMA]. Incluye: título principal, 4 secciones con icono + título + 2-3 líneas, un dato estadístico destacado en grande y fuente al pie. Estilo: moderno, paleta [azul y blanco / verde y negro], tipografía sans-serif, fondo no blanco.
🗂️
Whimsical Diagrams
Flujos, mapas mentales y wireframes desde texto
Gratis▾

Qué hace: Genera diagramas de flujo y mapas mentales editables en Whimsical a partir de descripciones en lenguaje natural.

Caso de uso: Visualizar algoritmos, procesos históricos, ciclos biológicos. Úsalo para flujos con decisiones (Sí/No) y GitMind (Módulo 07) para estructuras jerárquicas conceptuales.

prompt-whimsical.txt
# PROMPT PARA WHIMSICAL DIAGRAMS Crea un diagrama de flujo que represente el proceso de [PROCESO]. - Inicio y fin claramente marcados. - Al menos 2 decisiones (nodos Sí/No). - Máximo 10 nodos. Etiquetas en español.
🎓
Scholar AI
+200M papers académicos con DOIs verificables
Freemium▾

Qué hace: Accede a bases de datos académicas desde ChatGPT respondiendo con DOIs verificables. Nunca inventa referencias. Tiene ~3 consultas gratuitas por sesión; combínalo con Consensus (Módulo 08) para trabajo intensivo.

Caso de uso: Imprescindible para el TFB o Trabajo de Investigación.

prompt-scholar-ai.txt
# PROMPT PARA SCHOLAR AI Busca papers sobre [TEMA], publicados entre [AÑO] y [AÑO], revisados por pares. Para cada resultado: título, autores, año, DOI y resumen en español (máx. 3 frases). Sintetiza el estado del conocimiento en 100 palabras al final.
📐
Diagrams: Show Me
UML, ERD y arquitecturas técnicas en Mermaid
Gratis▾

Qué hace: Genera diagramas técnicos Mermaid (UML, ERD, secuencias, Gantt). Perfecto para la parte de programación y bases de datos del currículo TIC.

💡

Usa Show Me para diagramas UML/ER técnicos y Whimsical para flujos de proceso conceptuales.

prompt-showme.txt
# PROMPT PARA DIAGRAMS: SHOW ME Crea un diagrama entidad-relación (ERD) para una base de datos de [SISTEMA]. Entidades: [lista]. Incluye atributos, claves primarias, foráneas y cardinalidades.
Práctica — Flujo encadenado con 3 GPTs Práctica
  1. Scholar AI → Investigación base: busca 3-5 papers sobre tu tema. Copia títulos y resúmenes.
  2. Whimsical → Mapa mental: genera un mapa con los conceptos clave extraídos de los abstracts.
  3. Canva → Infografía: con el mapa como guía, genera una infografía lista para presentar.
  4. Verifica cada output: DOIs en Scholar AI, jerarquía en Whimsical, diseño en Canva.
  5. Cita las herramientas en tu trabajo igual que las fuentes bibliográficas.
