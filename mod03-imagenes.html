Explicación Teórica Teoría

Los modelos de generación de imágenes como Flux.1 (Black Forest Labs) son modelos de difusión: parten de ruido gaussiano y, iteración a iteración, lo refinan hasta generar una imagen coherente con el prompt. No "dibujan" de izquierda a derecha como un humano: convergen desde el caos hacia la coherencia.

Nano Banana es un frontend simplificado que permite acceder a varios modelos (incluyendo Flux) sin API key ni configuración técnica, ideal para el aula.

ℹ️

La calidad de la imagen depende en un 80% del prompt visual. A diferencia del texto, aquí importan: estilo artístico, iluminación, ángulo de cámara, paleta de color y resolución.

Práctica Guiada — Nano Banana + Flux.1 Práctica
  1. Accede a nanobanana.es y selecciona el modelo Flux.1 Schnell (versión rápida y gratuita).
  2. Escribe el prompt en inglés. Estructura: [Sujeto] + [Acción/Pose] + [Escenario] + [Estilo artístico] + [Iluminación] + [Calidad].
  3. Configura el Aspect Ratio según el destino: 1:1 para redes sociales, 16:9 para presentaciones.
  4. Genera y evalúa: ¿el estilo es correcto? ¿hay artefactos (manos deformadas, texto ilegible)?
  5. Itera el prompt: añade modificadores de estilo ("cinematic lighting, ultra-detailed, 8K") y compara resultados.
  6. Documenta el prompt exacto que produjo el mejor resultado. Un buen prompt visual es un activo reutilizable.
