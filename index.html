📄Prompt Engineering
Explicación Teórica Teoría

Un prompt es la instrucción que le das al modelo de lenguaje (LLM). La calidad de tu prompt determina directamente la calidad de la respuesta: entra basura, sale basura (garbage in, garbage out).

La Ingeniería de Prompts es la disciplina que estudia cómo formular instrucciones óptimas para obtener outputs precisos, repetibles y útiles. No es magia: es ingeniería de parámetros textuales.

El modelo no "entiende" como un humano. Predice tokens basándose en probabilidades. Tu prompt actúa como el contexto inicial que condiciona todo el espacio de respuestas posibles.

💡

Técnica R-P-C: La estructura más efectiva para un prompt educativo o profesional se construye con tres pilares: Rol (quién es la IA), Proceso (qué debe hacer y cómo) y Contexto (información relevante del entorno o la tarea).

ElementoPregunta claveEjemplo
R — Rol¿Quién es la IA?"Actúa como un experto en marketing digital..."
P — Proceso¿Qué debe hacer exactamente?"...redacta 5 titulares para Instagram en tono informal..."
C — Contexto¿Con qué información cuenta?"...para una tienda de ropa sostenible dirigida a jóvenes de 18-25 años."
Práctica Guiada Práctica
  1. Abre ChatGPT o Gemini en una nueva conversación. Empieza desde cero para no arrastrar contexto previo.
  2. Identifica el Rol: Decide qué experto necesitas. Cuanto más específico, mejor. "Experto" es vago; "Ingeniero de datos con 10 años en Python y Pandas" es preciso.
  3. Define el Proceso: Usa verbos de acción claros: resume, compara, genera, lista, explica, reescribe, traduce, extrae, clasifica. Especifica el formato de salida (tabla, lista, párrafos, JSON…).
  4. Añade el Contexto: Pega el texto, dato o situación relevante. Si el contexto es largo, indícale al modelo que espere antes de responder: "No respondas aún, primero lee esto:".
  5. Envía y evalúa: Si la respuesta no cumple, aplica prompt chaining: refina en el mismo chat añadiendo restricciones ("Más conciso", "Solo en español", "Sin tecnicismos").
  6. Documenta tu prompt en un bloc de notas. Un buen prompt es un activo reutilizable.
Prompt Maestro R-P-C Prompt

Copia este template y sustitúye los campos entre corchetes:

prompt-rpc-template.txt
# TÉCNICA R-P-C — PROMPT MAESTRO ROL: Actúa como [define el perfil experto: profesión, especialidad, años de experiencia]. Tu tono debe ser [formal / informal / técnico / divulgativo]. PROCESO: Tu tarea es [verbo de acción + descripción precisa de la tarea]. El formato de la respuesta debe ser [lista numerada / tabla / párrafos / JSON / markdown]. La extensión máxima es [X palabras / X párrafos / X ítems]. CONTEXTO: [Pega aquí el texto, dato, situación o restricciones relevantes. Si no hay contexto externo, describe el escenario con detalle.] RESTRICCIONES OPCIONALES: - Responde siempre en español. - No incluyas introducciones ni despedidas. - Si no sabes algo con certeza, indícalo explícitamente.

Claves de Elección Teoría

No existe "el mejor chatbot": existe el más adecuado para cada tarea. La elección correcta depende de tres variables: acceso a Internet en tiempo real, ventana de contexto (cuánto texto puede procesar a la vez) y coste.

⚠️

Privacidad de datos: Ningún chatbot gratuito garantiza que tus conversaciones no se usen para entrenamiento. No pegues datos personales, contraseñas ni información confidencial en ninguno de ellos.

Práctica Guiada — Benchmark personal Práctica
  1. Selecciona una tarea concreta de tu asignatura (ej: "Resume la Segunda Revolución Industrial en 5 puntos").
  2. Envía el mismo prompt a tres chatbots distintos simultáneamente (abre tres pestañas).
  3. Evalúa las respuestas en base a: precisión factual, extensión, citas de fuentes y legibilidad.
  4. Registra tus resultados en una tabla de Google Sheets con las columnas: Chatbot, Puntuación (1-5), Observaciones.
  5. Concluye qué chatbot resuelve mejor ESA tarea específica. Cambia de chatbot para tareas distintas.

¿Qué es un GPT? Teoría

Un GPT (en el contexto de ChatGPT) es un asistente personalizado construido encima del modelo base GPT-4o. La diferencia con el ChatGPT estándar es que el GPT lleva precargado un conjunto de instrucciones, herramientas externas y archivos de conocimiento específicos para un dominio concreto.

Técnicamente, un GPT es una combinación de tres capas: un system prompt extendido (instrucciones de rol), acciones (conexiones a APIs externas, como la de Canva o Whimsical) y knowledge files (documentos PDF propios que el creador del GPT ha subido como base de conocimiento).

ℹ️

Los GPTs son gratuitos para todos los usuarios de ChatGPT (con cuenta, sin suscripción de pago). La limitación es que el modelo subyacente en el plan gratuito es GPT-4o mini, no GPT-4o completo. Los resultados siguen siendo muy útiles para el aula.

Canva
Diseño gráfico con IA · Presentaciones, infografías, pósters
Gratis ▾

Qué hace: Conecta ChatGPT directamente con tu cuenta de Canva. Describes lo que quieres (una infografía sobre el cambio climático, un póster para la semana cultural, una presentación de 10 slides) y el GPT genera el diseño en Canva y te devuelve el enlace para editarlo.

Whimsical Diagrams
Diagramas de flujo, mapas mentales y wireframes desde texto
Gratis ▾

Qué hace: Genera diagramas de flujo, mapas mentales y wireframes directamente en Whimsical a partir de una descripción en lenguaje natural. El GPT convierte tu texto en un diagrama editable con un clic.

Caso de uso educativo: Perfecto para visualizar algoritmos, procesos históricos, ciclos biológicos, estructuras de datos o cualquier concepto que tenga relación causa-efecto o secuencia lógica.

ℹ️

A diferencia de GitMind (Módulo 07), Whimsical destaca en diagramas de flujo y wireframes además de mapas mentales. Usa Whimsical cuando necesites representar un proceso con decisiones (rombo → bifurcación) y GitMind para estructuras jerárquicas de conceptos.

prompt-whimsical-gpt.txt
# PROMPT PARA WHIMSICAL DIAGRAMS GPT ── OPCIÓN A: Diagrama de flujo ── Crea un diagrama de flujo en Whimsical que represente el proceso de [PROCESO O ALGORITMO]. El diagrama debe incluir: - Punto de inicio y fin claramente marcados - Al menos 2 decisiones (nodos con condición Sí/No) - Máximo 10 nodos en total - Etiquetas en español ── OPCIÓN B: Mapa mental ── Crea un mapa mental en Whimsical sobre [TEMA]. Nodo central: "[TEMA]" Ramas principales (5): [lista las ramas] Cada rama debe tener 3 subnodos concisos (máximo 3 palabras).

Scholar AI
Búsqueda y análisis de papers científicos con citas reales
Freemium ▾

Qué hace: Accede a bases de datos de más de 200 millones de artículos académicos (Semantic Scholar, PubMed, arXiv) directamente desde ChatGPT. Responde preguntas con citas reales y DOIs verificables, nunca inventa referencias.

Caso de uso educativo: Imprescindible para el Trabajo de Investigación (TdR / Trabajo de Fin de Bachillerato). Permite hacer búsquedas académicas avanzadas sin salir de la interfaz de ChatGPT y con la ventaja de poder pedir resúmenes y comparativas en el mismo hilo de conversación.

⚠️

Scholar AI tiene un límite de ~3 consultas gratuitas por sesión. Para trabajo intensivo de investigación, combínalo con Consensus (Módulo 08) e Inciteful, que sí son 100% gratuitos.

Qué hace: Genera diagramas técnicos en formato Mermaid y los renderiza visualmente: diagramas de clases UML, entidad-relación, secuencias, Gantt, arquitecturas de sistemas y más. Ideal para la parte de programación y bases de datos del currículo TIC.

Práctica Guiada — Flujo de trabajo con GPTs Práctica

Vamos a usar tres GPTs encadenados para producir un trabajo completo sobre un tema de investigación:

  1. Abre ChatGPT y accede al GPT Store desde el menú lateral izquierdo → "Explorar GPTs". Busca cada GPT por su nombre exacto.
  2. Scholar AI → Investigación base: Usa el prompt de Scholar AI para encontrar 3-5 papers sobre tu tema. Copia los títulos y resúmenes.
  3. Whimsical Diagrams → Mapa mental: Abre el GPT de Whimsical y pide un mapa mental basado en los conceptos clave que encontraste en Scholar AI. Pega los términos más repetidos en los abstracts.
  4. Canva GPT → Infografía final: Con el mapa mental como guía, usa el GPT de Canva para generar una infografía resumen lista para imprimir o presentar.
  5. Revisa y edita cada output. Los GPTs cometen errores: verifica los datos de Scholar AI con el DOI, ajusta el mapa mental si tiene errores de jerarquía, y personaliza el diseño de Canva.
  6. Documenta el flujo: Captura de pantalla de cada paso. En el trabajo final, debes citar las herramientas usadas igual que citas las fuentes.

Explicación Teórica Teoría

Los modelos de generación de imágenes como Flux.1 (Black Forest Labs) son modelos de difusión: parten de ruido gaussiano y, iteración a iteración, lo refinan hasta generar una imagen coherente con el prompt.

Nano Banana es un frontend simplificado que permite acceder a varios modelos (incluyendo Flux) sin necesidad de API key ni configuración técnica, ideal para el aula.

ℹ️

La calidad de la imagen generada depende en un 80% del prompt visual. A diferencia de los prompts de texto, aquí importan: estilo artístico, iluminación, ángulo de cámara, paleta de color y resolución.

ParámetroQué controlaValores comunes
PromptContenido y estiloDescripción detallada en inglés
Negative promptQué excluir"blurry, low quality, watermark"
Aspect ratioProporciones1:1, 16:9, 9:16 (stories)
StepsIteraciones de difusión20-30 (calidad) / 4-8 (velocidad)
CFG ScaleAdherencia al prompt4-7 (balance) / 8+ (literal)
Práctica Guiada — Nano Banana + Flux.1 Práctica
  1. Accede a nanobanana.es y selecciona el modelo Flux.1 Schnell (versión rápida y gratuita).
  2. Escribe el prompt en inglés. La estructura recomendada es: [Sujeto] + [Acción/Pose] + [Escenario] + [Estilo artístico] + [Iluminación] + [Calidad].
  3. Configura el Aspect Ratio según el destino: 1:1 para redes sociales, 16:9 para presentaciones.
  4. Genera la imagen y evalúa: ¿el estilo es el correcto? ¿hay artefactos visuales (manos deformadas, texto ilegible)?
  5. Itera el prompt: añade modificadores de estilo (ej: "cinematic lighting, ultra-detailed, 8K") o cambia el sujeto y compara resultados.
  6. Descarga la imagen en máxima resolución. Documenta el prompt exacto que produjo el mejor resultado.

Generación de Presentaciones

Flujos de trabajo con Gamma y PowerPoint AI.

📋 Ficha Técnica

🔗Gamma.app
🔗PowerPoint + Copilot
📄Outline → Deck
📄Diseño automático
Explicación Teórica Teoría

Gamma usa IA generativa para transformar un esquema textual (outline) en una presentación completa con diseño, tipografías e imágenes integradas. El flujo es: Prompt → Outline generado → Deck editado.

PowerPoint + Copilot (Microsoft 365) permite generar slides desde un documento Word, refinar contenido con prompts dentro de la app y auto-diseñar layouts con Designer. Requiere licencia M365.

💡

El error más común es pedir a la IA que "haga una presentación sobre X" sin más contexto. Cuanto más específico seas (audiencia, duración, nº de slides, tono), más utilizables serán las diapositivas generadas.

Práctica Guiada — Gamma Práctica
  1. Crea una cuenta gratuita en gamma.app. El plan free incluye 400 créditos iniciales (suficiente para el aula).
  2. Selecciona "Generar" → "Presentación". Elige el modo "Generar desde un prompt".
  3. Introduce el prompt detallado (usa la estructura del Prompt Maestro de abajo). Gamma generará primero un outline de diapositivas.
  4. Revisa y edita el outline antes de generar. Añade, elimina o reordena secciones. Este paso es crítico para el resultado final.
  5. Selecciona el tema visual (paleta de colores y tipografía) y genera el deck completo.
  6. Edita slide por slide: haz clic en cualquier elemento para modificarlo. Puedes pedir a la IA que "reescriba este párrafo en tono más formal" directamente en el panel lateral.
  7. Exporta en formato PDF o PowerPoint (.pptx) para entregar o seguir editando.
Prompt Maestro — Gamma Prompt
gamma-presentation-prompt.txt
# PROMPT MAESTRO PARA GAMMA.APP Crea una presentación profesional sobre [TEMA]. Detalles: - Audiencia: [estudiantes de bachillerato / clientes / directivos] - Duración estimada: [10 minutos / 5 minutos] - Número de slides: [8-10 slides] - Tono: [formal y técnico / divulgativo / motivador] - Idioma: español Estructura requerida: 1. Portada con título impactante y subtítulo 2. Índice / Agenda 3. [Sección 1 — concepto clave] 4. [Sección 2 — datos o estadísticas relevantes] 5. [Sección 3 — ejemplos o casos de uso] 6. [Sección 4 — ventajas y limitaciones] 7. Conclusiones en 3 puntos clave 8. Slide de cierre con llamada a la acción o reflexión final Estilo visual: moderno, minimalista, con uso de iconos y datos visualizados. Incluye una imagen relevante en cada slide siempre que sea posible.

IA en Hojas de Cálculo

Aplicación de IA en Excel y Google Sheets.

📋 Ficha Técnica

🔗Microsoft Excel
🔗Google Sheets
📄Copilot en Excel
📄Gemini en Sheets
Explicación Teórica Teoría

La IA aplicada a hojas de cálculo actúa en tres niveles: generación de fórmulas (describes en lenguaje natural lo que quieres calcular y la IA escribe la fórmula), análisis de datos (identificación de tendencias, anomalías y correlaciones) y automatización (generación de scripts en Apps Script o VBA).

Incluso sin acceso a Copilot (de pago), puedes usar ChatGPT o Gemini como asistente externo: describes el problema, pegas una muestra de los datos y obtienes la fórmula lista para copiar.

ℹ️

Google Sheets integra Gemini de forma nativa (botón "Preguntar a Gemini" en el panel lateral). Permite analizar el rango seleccionado, crear gráficos y explicar fórmulas existentes.

Práctica Guiada — Fórmulas con IA Práctica
  1. Abre Google Sheets con una tabla de datos (puede ser ficticia: notas de alumnos, ventas por mes, etc.).
  2. Abre Gemini en una pestaña paralela. Describe la estructura de tu tabla: columnas, filas, qué datos contiene.
  3. Pide la fórmula en lenguaje natural: "Necesito una fórmula que calcule el promedio de las celdas B2:B20 solo si el valor en la columna C es mayor que 50".
  4. Copia la fórmula generada y pégala en la celda correspondiente de Sheets. Verifica el resultado.
  5. Si hay errores, vuelve al chat y pega el mensaje de error exacto. La IA corregirá la fórmula.
  6. Genera un gráfico: selecciona el rango de datos, activa el panel Gemini y pide "Crea un gráfico de barras comparando las columnas A y B".
Prompt Maestro — Asistente de Fórmulas Prompt
spreadsheet-formula-prompt.txt
# PROMPT MAESTRO — ASISTENTE DE HOJAS DE CÁLCULO Actúa como un experto en [Google Sheets / Microsoft Excel]. 📋 Ficha Técnica

📄ASR · Doblaje IA · STT
Explicación Teórica Teoría

Las tres herramientas de este módulo atacan el problema del habla como dato desde ángulos distintos:

📜
Transkribus
Especializado en transcripción de documentos históricos y manuscritos. Usa modelos HTR (Handwritten Text Recognition). Ideal para Historia.
🎬
Rask.ai
Doblaje y traducción de vídeo con clonación de voz. Sube un vídeo en inglés, obtén una versión doblada al español manteniendo la voz original.
🖥️
Tactiq
Transcripción en tiempo real de reuniones (Google Meet, Zoom, Teams). Genera resúmenes automáticos y extrae tareas pendientes.
⚠️

Los modelos ASR (Automatic Speech Recognition) tienen una tasa de error (WER) que varía según el acento, la calidad del audio y el vocabulario técnico. Siempre revisa la transcripción antes de usarla como fuente.

Práctica Guiada — Tactiq en Google Meet Práctica
  1. Instala la extensión Tactiq desde la Chrome Web Store. Requiere una cuenta de Google.
  2. Inicia o únete a una sesión de Google Meet. Tactiq aparecerá automáticamente en el panel lateral.
  3. Activa la transcripción con el botón "Start Transcription". El texto aparecerá en tiempo real.
  4. Al finalizar la reunión, haz clic en "Generate Summary". Tactiq enviará la transcripción a un LLM y generará un resumen estructurado.
  5. Exporta el resumen a Google Docs o cópialo. Revisa que los nombres propios y términos técnicos sean correctos.

Organización Visual

Mapas mentales con GitMind.

📋 Ficha Técnica

🔗GitMind
📄Mapa mental · Diagrama de flujo
📄Generación desde texto
Explicación Teórica Teoría

GitMind es una herramienta de mapas mentales y diagramas que incorpora IA generativa para transformar texto en estructuras visuales jerárquicas. El flujo es: tema o texto → análisis semántico → árbol de nodos → exportación.

La utilidad pedagógica de los mapas mentales generados con IA reside en la externalización cognitiva: la IA extrae las relaciones entre conceptos que un alumno podría tardar horas en mapear manualmente.

💡

GitMind permite la edición colaborativa en tiempo real. Ideal para actividades de clase donde distintos grupos expanden ramas diferentes del mismo mapa.

Práctica Guiada — GitMind IA Práctica
  1. Crea una cuenta gratuita en gitmind.com con tu cuenta de Google.
  2. Selecciona "Nuevo" → "Generar con IA". Elige el tipo: Mapa Mental o Diagrama de Flujo.
  3. Introduce el topic o pega un texto. Puedes pegar un párrafo de un artículo y GitMind extraerá los nodos automáticamente.
  4. Revisa la estructura generada. La IA puede cometer errores de jerarquía: verifica que los nodos padre-hijo tengan sentido lógico.
  5. Añade o elimina nodos haciendo clic. Usa Tab para crear subnodos y Enter para nodos del mismo nivel.
  6. Cambia el tema visual (paleta de colores) para mejorar la legibilidad. Activa el modo "Presentación" para usarlo en clase.
  7. Exporta en formato PNG, PDF o comparte el enlace público para entregar al profesor.

MÓDULO 08

Investigación con IA

Consensus e Inciteful: búsqueda académica aumentada.

📋 Ficha Técnica

🔗Consensus
🔗Inciteful
📄Literatura académica · Grafo de citas
Explicación Teórica Teoría

Consensus es un motor de búsqueda académica que procesa más de 200 millones de papers científicos. A diferencia de Google Scholar, la IA sintetiza el consenso científico sobre una pregunta de investigación y lo presenta con porcentajes de acuerdo entre estudios.

Inciteful trabaja con el grafo de citas: dado un paper de partida, mapea los artículos que lo citan y los que cita, permitiendo descubrir el estado del arte de un campo de forma visual y no lineal.

⚠️

Pensamiento crítico obligatorio: Consensus resume estudios pero no sustituye su lectura. Un paper puede estar mal diseñado aunque sea reciente. Verifica siempre el tamaño de muestra, el año y la metodología antes de citar.

Práctica Guiada — Consensus + Inciteful Práctica
  1. Abre consensus.app y formula tu pregunta de investigación en inglés. Las preguntas tipo "Does X cause Y?" o "Is X effective for Z?" funcionan mejor.
  2. Analiza el Consensus Meter: el porcentaje de estudios que apoyan / contradicen / son neutros sobre tu hipótesis.
  3. Selecciona 2-3 papers relevantes de los resultados. Haz clic en el título para ver el abstract y los metadatos (año, revista, citas recibidas).
  4. Copia el DOI o el título del paper más relevante y abre inciteful.xyz.
  5. Pega el paper en Inciteful y observa el grafo de citas. Identifica los nodos con más conexiones: son los papers más influyentes del campo.
  6. Exporta la lista de papers relacionados y úsala como bibliografía base para tu trabajo de investigación.

MÓDULO 09

NotebookLM

Análisis de fuentes propias y generación de pódcast con IA de Google.

📋 Ficha Técnica

🔗NotebookLM
📄RAG · Grounding · Audio Overview
📄PDF, Docs, YouTube, webs
Explicación Teórica Teoría

NotebookLM (Google) implementa una arquitectura RAG (Retrieval-Augmented Generation): en lugar de responder desde los parámetros del modelo, ancla ("grounds") todas sus respuestas exclusivamente en los documentos que tú le proporcionas. Esto elimina las alucinaciones sobre el contenido de tus fuentes.

Puedes cargar: PDFs, documentos de Google Docs, URLs de páginas web, vídeos de YouTube y archivos de texto. El sistema indexa el contenido y permite interrogarlo con lenguaje natural.

La función "Audio Overview" genera un pódcast de 10-15 minutos con dos voces sintéticas que debaten y explican el contenido de tus fuentes. Es ideal para repasar antes de un examen.

ℹ️

Ventaja clave: NotebookLM siempre cita el fragmento exacto de la fuente de donde extrajo cada respuesta. Puedes hacer clic en la cita para saltar al texto original.

Práctica Guiada — Notebook de estudio Práctica
  1. Accede a notebooklm.google.com con tu cuenta de Google (gratuito sin límites de fuentes).
  2. Crea un nuevo Notebook para la asignatura o tema que estás estudiando.
  3. Añade fuentes: sube los PDFs de tus apuntes, añade la URL del libro de texto online o pega el enlace del vídeo explicativo de YouTube.
  4. Haz preguntas en el chat: "¿Cuáles son las diferencias entre ADN y ARN según mis apuntes?", "Resume el Tema 3 en un esquema".
  5. Verifica las citas: cada respuesta muestra el número de fuente entre corchetes [1]. Haz clic para leer el fragmento original y confirmar que la IA no está inventando.
  6. Genera el Audio Overview: haz clic en "Generate" en el panel de Audio. Espera 2-3 minutos. Escúchalo mientras revisas tus apuntes.
  7. Crea una "Guía de Estudio": en el panel de herramientas, selecciona "Study Guide". NotebookLM generará preguntas tipo examen con respuestas basadas en tus fuentes.
